# TMS API

This is the TMS API (Transport Management System).

## Installation and Running

To get started, you'll need to have [Task](https://taskfile.dev/installation/) installed.

### Available Commands

*   `task setup`: Creates a virtual environment and installs dependencies.
*   `task dev`: Runs the Flask app in development mode.
*   `task start`: Runs the app with Gunicorn.
